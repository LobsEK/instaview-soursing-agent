# Instaview - Sourcing Agent

AI-powered candidate sourcing agent for the Instaview recruitment platform.

## Local Development

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → "Add New Project"
3. Import the GitHub repo
4. Click "Deploy" — done!
